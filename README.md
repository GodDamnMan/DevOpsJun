
## Предварительные требования

- docker 
- docker compose 


## Бстрый старт

```bash
git clone https://github.com/GodDamnMan/DevOpsJun.git
cd DevOpsJun
docker compose up --build -d 
```

## Проверка работоспособности

Базовый вариант из терминала с установленным `curl`
```bash
curl http://localhost
```
Ответ должен быть таким
```bash
Hello from Effective Mobile!
```

Также вы можете вбить http://localhost в свой браузер и получить страницу с надписью `Hello from Effective Mobile!`


## Роутинг

### Схема локального роутинга

`Ваш запрос на localhost:80` ↴

`docker перенаправляет запрос в контейнер с nginx так как в нем открыт и проброшен порт 80` ↴

`контейнер с nginx принимает запрос и проксирует его в backend:8080` ↴

`контейнер с backend принимает запрос и отдает ответ который проходит такой же путь в обратную сторону`

---

### Схема глобального роутинга (Масштабирование)

`Ваш запрос на домен` ↴ 

`**интернет**` ↴

`Запрос доходит до машины на которой стоит docker` ↴

`docker перенаправляет запрос в контейнер с nginx так как в нем открыт и проброшен порт 80` ↴

`контейнер с nginx принимает запрос и проксирует его в backend:8080` ↴

`контейнер с backend принимает запрос и отдает ответ который проходит такой же путь в обратную сторону`
